# Django Authentication Project | Проект аутентификации на Django

## 🇬🇧 English Version

### 📌 Description
This project is a Django-based web application that implements user authentication. It includes user registration, login, logout, and a protected page accessible only to authenticated users.

The project was developed as part of a practical assignment for learning Django.

### 🚀 Features
- User registration
- User login and logout
- Protected page for authenticated users
- Redirect for unauthorized users to the login page
- Built using Django’s built-in authentication system

### 🛠️ Technologies Used
- Python 3
- Django
- HTML
- SQLite3
- Git & GitHub
